代码高亮插件，支持古腾堡代码块扩展。

== Description ==

io Code Highlight 使用 Prism Syntax highlighter 添加语法高亮显示支持。

Prism语法highlighter运行前端加载的JavaScript来解析代码块并应用语法标记。

== Installation ==

1. 上传 `io-code-highlight`目录 到 `/wp-content/plugins/` 目录
2. 激活插件。
